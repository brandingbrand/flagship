## Plugin Target Application

`@brandingbrand/kernel-plugin-target-application` is a package that configures all the necessary native code for generating iOS target applications.

### Getting Started

<div class="flex w-full my-6">
  <CodeCopy args="yarn add -D @brandingbrand/kernel-plugin-target-application" />
</div>

Add the above noted package and the plugin is avaialble in the Kernel plugin ecosystem.
