## Plugin Permissions

`@brandingbrand/kernel-plugin-permissions` is a package that configures all the necessary native code for [**react-native-permissions**](https://github.com/zoontek/react-native-permissions) library.

### Getting Started

<div class="flex w-full my-6">
  <CodeCopy args="yarn add -D @brandingbrand/kernel-plugin-permissions && yarn add react-native-permissions@^3.6.1" />
</div>

Add the above noted package and the plugin is avaialble in the Kernel plugin ecosystem.

### Usage

After adding this plugin as a dev dependency it is necessary to add it to the kernel plugins configuration via the `package.json`.

`package.json`

```
  "kernel": {
    "plugins": [
      "@brandingbrand/kernel-plugin-permissions"
    ]
  },
```

Your Kernel environment should also be updated with the `@brandingbrand/kernel-plugin-permissions` required props from the defined interface.

`.kernelrc/env/env.<env>.ts`

```
  kernelPluginPermissions: {
    kernel: {
      ios: [
        {
          permission: "LOCATION_WHEN_IN_USE",
          text: "App would like to use your location",
        },
        {
          permission: "NOTIFICATIONS",
        },
      ],
      android: ["ACCESS_FINE_LOCATION"],
    },
  },
```

The interface in your `.kernelrc/env/env.<env>.ts` has the following props:

<div class="my-4 overflow-scroll">
| attribute                            | description                                                             |
| :--------                            | :----------                                                             |
| kernel.ios                           | a list of required permissions with their associated usage descriptions |
| kernel.android                       | a list of required permissions                                          |
</div>

Valid permissions that can be used can be found here: [**ios**](https://github.com/brandingbrand/flagship/blob/482c1034b26db3fd98ae299428dcef6704914c52/packages/plugin-permissions/src/utils/ios.ts#L6) and [**android**](https://github.com/brandingbrand/flagship/blob/482c1034b26db3fd98ae299428dcef6704914c52/packages/plugin-permissions/src/utils/android.ts#L3).
