## Plugin Firebase Analytics

`@brandingbrand/kernel-plugin-firebase-analytics` is a package that configures all the necessary native code for [**@react-native-firebase/analytics**](https://github.com/invertase/react-native-firebase/tree/main/packages/analytics) library. Before using this plugin please configure the [**@brandingbrand/kernel-plugin-firebase-app**](/en/packages/plugins/firebase-app).

### Getting Started

<div class="flex w-full my-6">
  <CodeCopy args="yarn add -D @brandingbrand/kernel-plugin-firebase-analytics && yarn add @react-native-firebase/analytics" />
</div>

Add the above noted package and the plugin is avaialble in the Kernel plugin ecosystem.

### Usage

After adding this plugin as a dev dependency it is necessary to add it to the kernel plugins configuration via the `package.json`.

`package.json`

```
  "kernel": {
    "plugins": [
      "@brandingbrand/kernel-plugin-firebase-analytics"
    ]
  },
```

Your Kernel environment should also be updated with the `@brandingbrand/kernel-plugin-firebase-analytics` required props from the defined interface.

`.kernelrc/env/env.<env>.ts`

```
  kernelPluginFirebaseAnalytics: {
    kernel: {
      ios: {
        disableAdId: true,
      },
    },
  },
```

The interface in your `.kernelrc/env/env.<env>.ts` has the following props:

<div class="my-4 overflow-scroll">
| attribute                         | description                    |
| :--------                         | :----------                    |
| kernel.ios.disableAdId                   | disable advertising identifier |
</div>
