## Plugin Leanplum

`@brandingbrand/code-plugin-leanplum` is a package that configures all the necessary native code for [**@leanplum/react-native-sdk**](https://github.com/Leanplum/Leanplum-ReactNative-SDK/tree/1.2.1) library.

### Getting Started

<div class="flex w-full my-6">
  <CodeCopy args="yarn add -D @brandingbrand/code-plugin-leanplum && yarn add @leanplum/react-native-sdk@~2.0.0 clevertap-react-native@^0.9.4" />
</div>

Add the above noted package and the plugin is available in the Flagship Code plugin ecosystem.

### Usage

After adding this plugin as a dev dependency it is necessary to add it to the Flagship Code plugins configuration via the `package.json`.

`package.json`

```
  "code": {
    "plugins": [
      "@brandingbrand/code-plugin-leanplum"
    ]
  },
```

Your Flagship Code environment should also be updated with the `@brandingbrand/code-plugin-leanplum` required props from the defined interface.

`.coderc/env/env.<env>.ts`

```
  codePluginLeanplum: {
    code: {
      ios: {
        swizzle: false,
      },
      android: {
        leanplumFCMVersion: "5.10.1",
      },
    },
  },
```

The interface in your `.coderc/env/env.<env>.ts` has the following props:

<div class="my-4 overflow-scroll">
  | attribute | description | | :-------- | :---------- | | code.ios.swizzle |
  enable or disable swizzling | | code.android.leanplumFCMVersion | Leanplum fcm
  library version |
</div>
