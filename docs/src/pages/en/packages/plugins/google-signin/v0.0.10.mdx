## Plugin Google Signin

`@brandingbrand/kernel-plugin-google-signin` is a package that configures all the necessary native code for [**@react-native-google-signin/google-signin**](https://github.com/react-native-google-signin/google-signin) library. Before using this plugin please configure the [**@brandingbrand/kernel-plugin-firebase-app**](/en/packages/plugins/firebase-app).

### Getting Started

<div class="flex w-full my-6">
  <CodeCopy args="yarn add -D @brandingbrand/kernel-plugin-google-signin && yarn add @react-native-google-signin/google-signin@^8.0.0" />
</div>

Add the above noted package and the plugin is avaialble in the Kernel plugin ecosystem.

### Usage

After adding this plugin as a dev dependency it is necessary to add it to the kernel plugins configuration via the `package.json`.

`package.json`

```
  "kernel": {
    "plugins": [
      "@brandingbrand/kernel-plugin-google-signin"
    ]
  },
```

Your Kernel environment should also be updated with the `@brandingbrand/kernel-plugin-google-signin` required props from the defined interface.

`.kernelrc/env/env.<env>.ts`

```
  kernelPluginGoogleSignin: {
    kernel: {
      ios: {
        reversedClientId:
          "com.googleusercontent.apps.12345-abc123",
      },
      android: {
        googlePlayServicesAuthVersion: "19.2.0",
        swiperefreshlayoutVersion: "1.0.0",
      },
    },
  },
```

The interface in your `.kernelrc/env/env.<env>.ts` has the following props:

<div class="my-4 overflow-scroll">
| attribute                                    | description                                            |
| :--------                                    | :----------                                            |
| kernel.ios.reversedClientId                  | reverse client identifier                              |
| kernel.android.googlePlayServicesAuthVersion | google play services auth version                      |
| kernel.android.swiperefreshlayoutVersion     | androidx.swiperefreshlayout:swiperefreshlayout version |
</div>
