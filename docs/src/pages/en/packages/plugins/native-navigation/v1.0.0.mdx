## Plugin Native Navigation

`@brandingbrand/code-plugin-native-navigation` is a package that configures all the necessary native code for [**react-native-navigation**](https://github.com/wix/react-native-navigation) library.

### Getting Started

<div class="flex w-full my-6">
  <CodeCopy args="yarn add -D @brandingbrand/code-plugin-native-navigation && yarn add react-native-navigation@^7.31.1" />
</div>

Add the above noted package and the plugin is available in the Flagship Code plugin ecosystem.

### Usage

After adding this plugin as a dev dependency it is necessary to add it to the Flagship Code plugins configuration via the `package.json`.

`package.json`

```
  "code": {
    "plugins": [
      "@brandingbrand/code-plugin-native-navigation"
    ]
  },
```
