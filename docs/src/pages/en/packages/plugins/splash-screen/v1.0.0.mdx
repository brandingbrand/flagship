## Plugin Splash Screen

`@brandingbrand/code-plugin-splash-screen` is a package that configures all the necessary native code for a native splash screen.

### Getting Started

<div class="flex w-full my-6">
  <CodeCopy args="yarn add -D @brandingbrand/code-plugin-splash-screen" />
</div>

Add the above noted package and the plugin is available in the Flagship Code plugin ecosystem.

### Usage

After adding this plugin as a dev dependency it is necessary to add it to the Flagship Code plugins configuration via the `package.json`.

`package.json`

```
  "code": {
    "plugins": [
      "@brandingbrand/code-plugin-splash-screen"
    ]
  },
```

Your Flagship Code environment should also be updated with the `@brandingbrand/code-plugin-splash-screen` required props from the defined interface.

`.coderc/env/env.<env>.ts`

```
  codePluginSplashScreen: {
    code: {
      ios: {
        type: 'generated',
        generated: {
            logoPath: "assets/splash-screen/ios/generated/logo.png",
            size: 212,
            backgroundColor: "#ffffff"
        },
      },
      android: {
        type: 'generated',
        generated: {
            logoPath: "assets/splash-screen/ios/generated/logo.png",
            size: 180,
            backgroundColor: "#ffffff"
        },
      },
    },
  },
```

or

```
  codePluginSplashScreen: {
    code: {
      ios: {
        type: "legacy",
        legacy: {
          path: "assets/splash-screen/ios/legacy",
          xcassets: "LaunchImages.xcassets",
        },
      },
      android: {
        type: "legacy",
        legacy: {
          path: "assets/splash-screen/android/legacy",
        },
      },
    },
  },
```

The legacy splash screen option can be utilized if you have all the generated assets - this schema is preferred if you have a complex splash screen i.e. background images, off-centered offsets, etc.

The generated splash screen option can be used if you have a very simple splash screen with a background color and a centered logo.

The interface in your `.coderc/env/env.<env>.ts` has the following props:

<div class="my-4 overflow-scroll">
| attribute                                | description                                  |
| :--------                                | :----------                                  |
| code.ios.type                          | generated or legacy type                     |
| code.ios.legacy.path                   | path to the legacy assets                    |
| code.ios.legacy.xcassets               | name, including extension, of xcode xcaseets |
| code.ios.generated.logoPath            | path to logo for generated option            |
| code.ios.generated.size                | size of logo to be generated                 |
| code.ios.generated.backgroundColor     | background color of splash screen            |
| code.android.type                      | generated or legacy type                     |
| code.android.legacy.path               | path to legacy assets                        |
| code.android.generated.logoPath        | path to logo for generated option            |
| code.android.generated.size            | size of logo to be generated                 |
| code.android.generated.backgroundColor | background color of splash screen            |
</div>
