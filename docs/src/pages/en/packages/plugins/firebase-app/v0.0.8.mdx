## Plugin Firebase App

`@brandingbrand/kernel-plugin-firebase-app` is a package that configures all the necessary native code for [**@react-native-firebase/app**](https://github.com/invertase/react-native-firebase/tree/main/packages/app) library.

### Getting Started

<div class="flex w-full my-6">
  <CodeCopy args="yarn add -D @brandingbrand/kernel-plugin-firebase-app && yarn add @react-native-firebase/app" />
</div>

Add the above noted package and the plugin is avaialble in the Kernel plugin ecosystem.

### Usage

After adding this plugin as a dev dependency it is necessary to add it to the kernel plugins configuration via the `package.json`.

`package.json`

```
  "kernel": {
    "plugins": [
      "@brandingbrand/kernel-plugin-firebase-app"
    ]
  },
```

Your Kernel environment should also be updated with the `@brandingbrand/kernel-plugin-firebase-app` required props from the defined interface.

`.kernelrc/env/env.<env>.ts`

```
  kernelPluginFirebaseApp: {
    ios: {
      podfile: {
        pods: [
          "pod 'Firebase', :modular_headers => true",
          "pod 'FirebaseCore', :modular_headers => true",
          "pod 'GoogleUtilities', :modular_headers => true",
          "$RNFirebaseAsStaticFramework = true",
        ],
      },
    },
    kernel: {
      ios: {
        googleServicesPath: "assets/firebase/GoogleService-Info.dev.plist",
      },
      android: {
        googleServicesPath: "assets/firebase/google-services.dev.json",
        googleServicesVersion: "4.3.3",
        firebaseBomVersion: "30.4.1",
      },
    },
  },
```

The interface in your `.kernelrc/env/env.<env>.ts` has the following props:

<div class="my-4 overflow-scroll">
| attribute                            | description                                     |
| :--------                            | :----------                                     |
| ios.podfile.pods                     | additional ios pods necessary for firebase      |
| kernel.ios.googleServicesPath        | path to the GoogleService-Info.plist            |
| kernel.android.googleServicesPath    | path to the google-services.json                |
| kernel.android.googleServicesVersion | google services version                         |
| kernelandroid.firebaseBomVersion     | firebase BOM version                            |
</div>
